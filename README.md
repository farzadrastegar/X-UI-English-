#### How to Run

Tested on Debian11 x64 in Azure (using the Debian 11 "Bullseye" image).

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install curl socat -y

wget https://raw.githubusercontent.com/farzadrastegar/X-UI-English-/master/install.sh
sudo chmod +x install.sh
sudo ./install.sh
```

For more information see [here](./README_ORIG.md).
